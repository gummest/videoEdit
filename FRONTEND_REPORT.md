## API Integration

**Endpoint:** `POST /api/process`

**Request:**
```
Content-Type: multipart/form-data
Fields:
  - video: <file>
  - totalLength: <number>
  - cutDuration: <number>
```

**Response:**
```
Content-Type: video/mp4 (blob)
```
